# 📄 Clause Landing Page (Dribbble Recreation)

A responsive landing page built with Next.js, inspired by a Dribbble design by Dipa Inhouse. This project focuses on translating a high-fidelity UI concept into a clean, functional frontend with attention to layout precision, responsiveness, and subtle UI animations.

## 🎨 Design Reference
- Design: https://dribbble.com/shots/23713828-Clause-Contract-Management-Landing-Page  
- Designer: Dipa Inhouse (https://dipainhouse.com)

---

## 🛠️ Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Canva (for asset recreation)

---

## ✨ Features
- Fully responsive layout (mobile, tablet, desktop)
- Pixel-accurate UI recreation from design reference
- Animated logo carousel for brand section
- Clean and modern navigation layout
- Reusable component structure
- Optimized layout for large screens (1440px+)

---

## 🧠 What I Focused On
This project was less about complex logic and more about execution quality.

- Translating static design → real UI  
- Maintaining consistent spacing, typography, and hierarchy  
- Handling responsiveness across breakpoints  
- Recreating missing assets manually  
- Adding subtle motion to improve UX  

---

## 🖼️ Assets
All design assets were recreated in Canva to match the original Dribbble concept as closely as possible.

---


## ⚙️ Getting Started

Clone the repo:

git clone https://github.com/ahriannacareer/clause.git
cd clause-landing-page

Install dependencies:

npm install

Run the dev server:

npm run dev

---

## 📱 Responsive Design
The layout adapts across:
- Mobile (stacked layout)
- Tablet (adjusted spacing and scaling)
- Desktop (centered max-width container)
- Large screens (optimized for 1440px+)
